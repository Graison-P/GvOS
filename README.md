# GvOS

GvOS is a customized version of LineageOS that has the goal of being 2 OSes in one. GvOS allows both Android apps and aswell as a custom app format that allows both Android or Linux apps, which also means one side (eg Linux) can communicate to the other. (eg LineageOS).
GvOS is also the best for games and media! (With the ability to also install from the Google Play Store, aswell, with MindTheGapps support.)
