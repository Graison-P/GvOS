# System Sounds Directory

Place custom `.ogg` or `.wav` audio files here for system notification sounds, boot sounds, and event sounds.

## Recommended Specifications

- **Format**: OGG Vorbis (preferred) or WAV
- **Sample Rate**: 44100 Hz
- **Channels**: Stereo
- **Duration**: 
  - Notification sounds: 1-3 seconds
  - Boot/shutdown sounds: 3-10 seconds

## Usage Examples

- `boot.ogg` - System startup sound
- `shutdown.ogg` - System shutdown sound
- `notification.ogg` - General notification sound
- `error.ogg` - Error alert sound
- `message.ogg` - Message received sound

Place your custom sound files here and reference them in your configuration or hook scripts.
