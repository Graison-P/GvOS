# GvOS Wallpapers

This directory contains default wallpapers for GvOS.

## Wallpaper Files

- `gvos-wallpaper-1.png` - Default GvOS wallpaper (primary)
- `gvos-wallpaper-2.png` - Alternative GvOS wallpaper

## Usage

These wallpapers are available for selection in the XFCE desktop environment settings.
The primary wallpaper (`gvos-wallpaper-1.png`) is set as the default on first boot.

## Customization

To add more wallpapers:

1. Copy your image files (PNG, JPG, or JPEG) to this directory
2. Update the XFCE desktop settings configuration if you want to set a different default

Recommended image specifications:
- Format: PNG or JPEG
- Resolution: 1920x1080 or higher
- Aspect ratio: 16:9 for best compatibility
