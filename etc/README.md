# GvOS /etc Directory

This directory contains system configuration files for GvOS.

## Configuration Files

### Core System Files
- `fstab` - Filesystem mount table
- `hostname` - System hostname
- `hosts` - Static hostname to IP address mappings
- `os-release` - Operating system identification

### Network Configuration
- `network/interfaces` - Network interface configuration

### Boot Configuration
- `default/grub` - GRUB bootloader settings

### Directory Structure
- `init.d/` - System initialization scripts (SysV init)
- `systemd/` - Systemd service configuration
- `network/` - Network configuration files

## Customization

These are placeholder files that demonstrate the expected configuration structure.
Modify these files according to your specific GvOS installation requirements.
